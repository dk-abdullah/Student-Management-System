# Student Management System

FastAPI Student Management System with JSON Database and Streamlit Frontend.

## Project Description

A full-stack Student Management System built using **FastAPI** for backend and **Streamlit** for frontend.  
The system performs CRUD operations on student data stored in a JSON file.

## Features

### Backend (FastAPI)
- View all students  
- View student by ID  
- Add new student  
- Update student details  
- Delete student  
- Sort students by Semester and Section  

### Frontend (Streamlit)
- View students in UI  
- Add student form  
- Search student by ID  
- Update student details  
- Delete student
